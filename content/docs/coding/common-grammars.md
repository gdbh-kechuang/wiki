## 注释

注释（comment）是写在源代码里供人类阅读的内容，多用作解释代码原理、注意事项之类的。可以单独一行，也能跟在一般语句後面。还有多行的注释。

常见的形式是这样： `//` 、 `#` 、 `/**/`

```python
print('hello from python!')     # 输出一句 hello from python!
```

```c
/*
这是一个多行注释
用于说明代码的功能和目的。
你可以在这里面添加任何你想要的说明信息。
 */
```

## 语句终止符

大多数语言使用分号 `;` 作为一般语句的结束标记。有些语言允许省略终止符。

```c
// 定义一个函数，计算两个数的和
int add(int a, int b) {
    return a + b;
}
```

也有其他语言用换行、缩进等方式整顿语句。

```python
def multiplication_table(n):
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            product = i * j
            print(f"{product}", end=' ')
        print()
```

## 创建数据

### 标识符

计算机处理很多很多数据，这些数据被放置在一定的空间，而这些空间当然要有名字才能被正确识别，这些名字更严谨的叫法是标识符（identifier）。

标识符的命名很自由，但不是能完全随便起的，许多语言都有这些限制：
- 不得使用保留字（Reserved word），这些是一门语言用作语法的固有字词；
- 不得带有特殊符号，常见有空格 ` ` 、百分号 `%` 、哈希符号 `#` 等等；
- 不得以数字开头；
- 不超出长度限制。

### 声明、定义、赋值

为了保存数据，我们需要声明（declaration）并定义（definition）一个变量。如此就能用来存储数据了。下面这两句创建了名为“x”和“y”的变量，随后分别把10和20赋值（assignment）给了x和y。

```c
int x;      // 声明
x = 10;     // 定义

/* 声明和定义可以写作一行。 */
int y = 20;
```

有些语言不明确区分声明和定义。声明即是定义，定义即是声明。

```python
# python中声明并定义一个变量a，如果单写一个a会报错标识符不存在。
a = 'www'
```

### 变量、常量、字面量、修饰符

和数学中的变量（variable）一样，变量可以存储任何合法的值，至于什么值合法，参考各语言的[数据类型](#类型系统)。多数语言的声明语句，默认创建的是一个变量。

## 类型系统

数字类的数据可以进行数学运算，但是对于文本类的数据，数学运算就没什么意义。因此各种数据就应该区分成不同的数据类型（Data type），对不同数据采取不同处理方式。

显而易见，有些语言在声明变量的时候还需要在标识符附件带上`int`、`float`之类的关键字，这就是指明该变量存储的是什么类型的数据。但不是所有语言都需要指明类型，这取决于这门语言是**静态的**还是**动态的**。

动态语言的变量在运行时能转换成其他类型；

```javascript
let dynamicVar = 'hello from js';    // 这个变量是字符串
dynamicVar = 233;    // 现在是数字
```

静态语言的变量，在定义时就确定了其类型，後续不能改变。

```kotlin
var staticVar : Int = 233 // 定义了一个整型变量

/* Assignment type mismatch: actual type is 'kotlin.String', but 'kotlin.Int' was expected. */
staticVar = "233" // 爆！整型变量不能赋其他类型的值

```
