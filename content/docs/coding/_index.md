## 编程语言是？

编程语言（programming language）是用来定义计算机程序的形式化符号，人类能通过这种东西让计算机执行定义好的程序，辅助我们解决问题。

一门编程语言并不仅仅是写代码那么单纯而已，编程语言可以大致分为三部分：语法规则、运行时环境、社区生态。

而这一板块本先讨论语法规则，介绍不同编程语言的形式符号背後共通的思想，避免每次新学一门语言都在重复的语法知识点浪费生命。接下来的内容使用几种主流编程语言（C、Python、JS、Kotlin等）讨论语法知识，方便观察各种语言的相似之处。而各语言的具体语法当然不是重点，概不讨论。
