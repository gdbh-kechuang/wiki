## 编程语言是？

编程语言（programming language）是一种计算机程序的形式化符号，人类使用这种形式语言描述“程序”，交由计算机执行来辅助人类解决问题。

一门编程语言不仅仅是在文本编辑器里写代码那么单纯，编程语言也是软件，大致分为三部分：语法规则、运行环境、社区生态。

- 语法规则：编程语言的开发者制定的编程语言的代码语法
- 运行环境：编程语言的“本体”，也是由开发人员所创造、维护
- 生态社区：围绕一门编程语言随时间发展而来的社区，主角是各位开发者和各种软件
